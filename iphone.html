private tutoring

Using projects based on level (beginner, intermediate, advanced) that cover specific programming concept(s), students learn in groups of 1-3 once a week on Zoom completing these projects together. This allows students to learn how to debug and improve code collaboratively.

Parents are updated biweekly via email on their child's performance.

Beginner students start with Scratch, a friendly user interface to learn the concepts/logic of coding.

This stage is reccomended for at least two months.

Intermediate students work with Python, one of the most popular languages today. If the student enjoys creating artistically, I offer HTML/CSS/JS tutoring at this step, which is used to make websites/apps.

Advanced students work in Java to prepare for AP Computer Science A, as well as practice using custom libraries such as Ren'Py and PyGame to practice their skills.

At this step, I help students create their own Github profile, which is extremely important when seeking employment.
